# v0.4 Vue Frontend

## 目标

新增 Vue 管理端，复用同一套后端 API，用第二种前端技术栈实现登录和用户管理基础页面。

## 新增能力

Vue 前端。

## 技术范围

| 类别 | 技术 |
|---|---|
| 前端 | Vue |
| 通信 | HTTP |
| 后端 | 复用 `v0.2` API |

## 页面范围

| 页面 | 说明 |
|---|---|
| 登录页 | 登录并保存 token |
| 首页 | 展示当前用户 |
| 用户列表 | 分页查询用户 |
| 用户编辑 | 新增或编辑用户 |

## 开发步骤

1. 创建 `frontend-vue` 项目。
2. 配置开发代理到后端 API。
3. 实现 API 客户端。
4. 实现登录状态管理。
5. 实现登录页和路由守卫。
6. 实现用户列表和编辑表单。
7. 对比 React 与 Vue 的目录组织和状态管理差异，记录到 README 或单独文档。
8. 更新 `docs/PROGRESS.md`。

## 验收标准

1. Vue 应用可以启动。
2. 浏览器可以登录。
3. 登录后可以查看当前用户。
4. 可以查看用户列表。
5. 可以新增或编辑用户。
6. React 前端仍然可用。
7. 后端 API 未因适配 Vue 产生破坏性变更。

## 完成定义

1. Vue 前端可运行。
2. 与后端接口联调通过。
3. README 记录 Vue 前端启动方式和端口。
4. 新增和修改代码已补充详细中文注释。
5. 如果使用 Git，提交并打 tag：`v0.4-vue-frontend`。
